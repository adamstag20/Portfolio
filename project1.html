CheckTheFridge

Created using ReactJS and ASP.NET

This is an application I worked on in my Software Engineering class with a group of fellow students. Its an app that allows you to create a record of ingredients you currently have available and to use the list of ingredients to match to recipes that use them. The goal is to help the user come up with recipe ideas that use the foods they have available so they are less likely to waste.

My role in the project was to build the interface for the login page and logic behind user access to the website via a user token. I also created the browsing recipe page that allows you to search recipes and see what ingredients you have available for them.
